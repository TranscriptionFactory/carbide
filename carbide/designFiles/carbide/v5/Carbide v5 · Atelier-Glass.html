9_CODE / metabolomicPriors / docs · 2026-06-04

Pathway-graph priors do not improve metabolite-embedding disease classification

● unsaved1,240 wordsbacklinks 4#proj/priors

The hypothesis was clean: if disease signal lives in coordinated pathways rather than single metabolites, then biasing the embedding with a pathway-graph prior should sharpen the downstream classifier. Three weeks of runs say otherwise.

Result

Across both NAFLD cohorts, the pathway-priored model is statistically indistinguishable from the unpriored baseline — AUROC deltas sit inside the seed-to-seed noise band, and the rank-bottleneck described in [[rank_bottleneck_S1]] persists regardless of the prior.

The prior changes which metabolites cluster, but not how separable the disease classes become. It reorganises without informing.

Why this matters

It closes the graph-prior branch for [[metaboloformer]] and frees effort for the quantified-ReDU pilot. The negative result is worth a clean writeup — see [[PRACTICAL_NEXT_STEP]].
